# Cobblestone Legacy

Rogue-lite sandbox RPG set in a procedurally generated city.

- Engine: Godot 4.4 (Jolt) + C++ GDExtension, SQLite
- Pillars: Survival, Heist/Stealth, Factions, Dynamic Narrative
- Community: Join our Discord (link in navbar)

See the **[Roadmap](/docs/roadmap)** and the **[Devlog](/blog)**.
